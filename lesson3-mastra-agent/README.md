# Lesson 3: Mastraで書き直す

Memory あり / Loop あり / Tool あり（Mastraで実装）

## 指示

```
今のシンプルなエージェントをMastraで書き直してください
Mastra v1.0 以上、AI SDK v6.0 以上を使用。
汎用的なエージェントにしつつ、getCurrentTime ツールを使って、「今何時？」と聞いたら時刻を返すように。

注意
- `@mastra/core/agent` と `@mastra/core/tools` からインポート
- モデルは `@ai-sdk/google` の `google()` 関数で作成
```
